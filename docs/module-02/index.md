# Module 2: Core Concepts

Welcome to Module 2! In this module, we dive deeper into the core concepts and algorithms that form the foundation of modern robotics.

## Module Overview

This module covers:

1. **Robot Kinematics** - Forward and inverse kinematics
2. **Robot Dynamics** - Forces, torques, and motion equations
3. **Path Planning** - Navigation and trajectory planning
4. **Control Systems** - PID, MPC, and adaptive control
5. **SLAM** - Simultaneous Localization and Mapping

## Prerequisites

Before starting this module, ensure you have completed [Module 1: Foundations](/docs/module-01).
